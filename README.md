## 下载

### Docker

```
docker pull apachecn0/apachecn-php-zh
docker run -tid -p <port>:80 apachecn0/apachecn-php-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g apachecn-php-zh
apachecn-php-zh <port>
# 访问 http://localhost:{port} 查看文档
```
